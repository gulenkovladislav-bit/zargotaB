Структура:
  data.json — мир (каталог, бестиарий, атлас) без картинок
  manifest.json — версия, счётчики
  user_data_migration.json — твои персонажи, арена и т.п. (одноразовый импорт при первом запуске)
  images/ — все картинки в правильной структуре

Использование:
  1. Распакуй в корень Git-репозитория
  2. Закоммить и запушь
  3. Открой новую версию Зарготы — она сама заметит user_data_migration.json и импортирует его
  4. После успешного импорта user_data_migration.json можно удалить
