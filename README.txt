Что делать:
1. Распакуй этот архив
2. Перетащи data.json и manifest.json в папку Git-репо (замени старые)
3. Перетащи содержимое папки images/ в images/ репо (добавит новые файлы)
4. В GitHub Desktop → Commit + Push

Содержимое:
  data.json           — мир (каталог, бестиарий, атлас)
  manifest.json       — версия + счётчики
  images/             — 1 новых картинок
